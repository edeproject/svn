EDE Buildbot

This is the central place where various information about EDE builds are reported. These builds are scheduled to be executed daily, pulling latest changes from our repository.

The major goal of using this Buildbot is to provide build coverage for edelib and ede on various OS-es with different (if possible) compilers.

A short explaination of the menu items

  • waterfall display - gives you a time-oriented summary of recent buildbot activity
  • grid display - gives you a developer-oriented summary of recent buildbot activity
  • latest build - information about latest build per builder
  • recent builds - information about latest builds are summarized here, one per line
  • buildslaves - information about connected peers and their capabilities
